# Excel Data Automation Tool (Python)

This Python tool automates Excel data cleaning, formatting, and processing tasks.

## What this tool does
- Cleans messy Excel files
- Removes duplicates and empty rows
- Formats columns automatically
- Handles large Excel files efficiently

## Who this is for
- Businesses working with Excel reports
- Freelancers handling client data
- Anyone who wants to save time on Excel tasks

## Tech Stack
- Python

- openpyxl
